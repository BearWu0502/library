
- 安裝nodeJS，版本要求：14.21.3 → 一定要這個版本，可以用nvm切換NodeJS版本
- 前後端需要分別開一個terminal

### back-end

```
**Terminal**
npm install
安裝完
npm run start
```

### front-end

```
**Terminal**
npm install
安裝完
npm run serve
```

- 開啟 http://localhost:8080/
